网络编程和文件输入输出有很多相似之处
网络编程就是编写程序使两台连网的计算机相互交换数据
两台计算机之间的数据传输，物理连接（无需考虑）
数据传输软件，操作系统提供名为socket的部件
socket是网络数据传输用的软件设备

为了与远程的计算机进行数据传输，需要连接因特网，为了连接因特网，就需要socket
socket分为两种:TCP socket




int socket(int dotmain,int type,int protocol);//创建TCP  socket
int bind(int sockfd,struct sockaddr*myaddr,socklen_t addrlen);//给socket分配地址信息（IP地址和端口号）
int listen(int sockfd,int backlog);//将socket转换成可连接的状态
int accept(int sockfd,struct sockaddr*addr,socklen_t*addrlen);//受理连接请求
int connect(int sockfd,struct sockaddr*serv_addr,socklen_t addrlen);//请求连接
